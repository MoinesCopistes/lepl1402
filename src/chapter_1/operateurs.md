# Operateurs

## Exposant

Pour créer un exposant comme $$a^b$$ on utilise 
```java
Math.pow(a,b);
```

## Racine carrée

Pour un racine carrée on utilisera 
```java
Math.sqrt(a);
```

## Division

En utilisant la division avec des entiers, le résultat sera aussi un entier, par exemple
```java
7/2 //-> 3
```
